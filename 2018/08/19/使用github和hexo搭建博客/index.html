使用github和hexo搭建博客

  主要参考了博客“使用Hexo+Github一步步搭建属于自己的博客（基础）（https://www.cnblogs.com/fengxiongZz/p/7707219.html）”

另外，在安装时，碰到了一些问题，解决如下：

  1. Node.js的安装与环境配置

解决：参考了博客“Node.js的安装及环境配置之Windows篇（https://www.cnblogs.com/zhouyu2017/p/6485265.html）”

  1. hexo init时，会在cmd环境下使用git命令，比如git clone
    解决： 在安装git时需要选择use Git from the Windows Command Prompt，应该是配置环境变量

  2. 在用hexo d -g同步时，提示错误”fatal bad indentation of a mapping entry”
    解决：在deployment配置_config.yml文件时，冒号后需要留一个空格
